🔤 НАПРАВЛЕНИЕ ТЕКСТА:

Примечание: Иврит пишется справа налево (RTL).
В текущей версии базовая поддержка RTL есть.

Для полноценной поддержки RTL добавьте в mechanic.html:

<html lang="he" dir="rtl">  ← когда lang=he

И в CSS:
[dir="rtl"] {
    text-align: right;
}

═══════════════════════════════════════════════════════════════

✨ ДОПОЛНИТЕЛЬНЫЕ УЛУЧШЕНИЯ (опционально):

Если нужна полная поддержка RTL для иврита:

1. Добавьте атрибут dir="rtl" когда lang=he
2. Измените выравнивание текста
3. Зеркально отразите некоторые элементы

Пример в mechanic.html:

<html lang="{{ lang }}" dir="{{ 'rtl' if lang == 'he' else 'ltr' }}">

И в CSS:
[dir="rtl"] .form-group label {
    text-align: right;
}

[dir="rtl"] .checkbox-item {
    flex-direction: row-reverse;
}
