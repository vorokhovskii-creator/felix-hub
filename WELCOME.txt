📝 ТЕСТИРОВАНИЕ:

1. Откройте: http://localhost:8000
2. Выберите язык (например, Русский)
3. Создайте тестовый заказ:
   - Имя: Давид
   - Гос номер: 123-45-678
   - Категория: Тормоза
   - Детали: Передние колодки
   - Тип: Аналог
4. Нажмите "Отправить заказ"
5. Зайдите в админ-панель (пароль: felix2025)
6. Измените статус на "Готово"
7. Чек автоматически распечатается в консоли!

🔔 TELEGRAM-УВЕДОМЛЕНИЯ (опционально):

Чтобы включить уведомления в Telegram:

1. Создайте бота через @BotFather
2. Получите токен бота
3. Узнайте свой Chat ID через @userinfobot
4. Отредактируйте файл .env:
   
   TELEGRAM_BOT_TOKEN=ваш_токен_бота
   TELEGRAM_ADMIN_CHAT_ID=ваш_chat_id

5. Перезапустите приложение (Ctrl+C, затем python app.py)
